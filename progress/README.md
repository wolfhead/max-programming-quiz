# Progress Tracking

这个目录用于记录学生学习进度，帮助后续出题贴合节奏。

建议每次完成题目后记录：

- 日期
- 题号
- 目标能力
- 完成情况
- 卡住的位置
- 下一题建议

## 示例

| Date | Quiz | Skills | Result | Notes | Next |
| --- | --- | --- | --- | --- | --- |
| 2026-07-05 | quiz_001 | modulo, even numbers, return value | pending | Not attempted yet | Practice `%` and `if` |

## 能力阶梯

建议按学生掌握情况逐步推进：

| Stage | Skills | Example Quiz |
| --- | --- | --- |
| 1 | return value, arithmetic expression | quiz_template |
| 2 | condition, modulo, even numbers | quiz_001 |
| 3 | vector, index, loop, adjacent values | quiz_002 |

## 记录格式

后续可以按日期追加记录：

```text
Date:
Quiz:
Result:
What went well:
Where stuck:
Next focus:
```
